# 策略模块化说明

当前版本已经把日线策略和分钟级B点策略都拆到 `strategies/` 目录下。

## 目录结构

```text
strategies/
├── __init__.py
├── base_strategy.py       # 日线策略和分钟策略的基础类
├── daily_strategies.py    # 日线策略实现
├── minute_strategies.py   # 分钟级B点策略实现
└── registry.py            # 策略注册表，统一管理启用哪些策略
```

## 日线策略

日线策略在 `strategies/daily_strategies.py` 中，目前包括：

1. 箱体突破
2. 底部放量反转
3. 主升-箱体突破
4. 主升-底部放量反转
5. 主升-缩量回调启动
6. 主升-均线多头排列

新增日线策略的方法：

1. 在 `daily_strategies.py` 中新增一个类，继承 `BaseDailyStrategy`。
2. 设置 `name` 和 `category`。
3. 实现 `match(self, row)`。
4. 在 `registry.py` 的 `get_daily_strategies()` 中注册。

## 分钟级B点策略

分钟级策略在 `strategies/minute_strategies.py` 中，目前包括：

1. 5分钟回踩均线启动，适合主升趋势类。
2. 5分钟平台突破确认，适合突破类，也作为其他分组的保底策略。
3. 5分钟放量反包确认，适合放量启动类。

30分钟结构确认 `check_30m_structure()` 是所有分钟B点前的公共过滤条件。

新增分钟策略的方法：

1. 在 `minute_strategies.py` 中新增一个类，继承 `BaseMinuteStrategy`。
2. 设置 `name` 和 `support_groups`。
3. 实现 `match(self, row, df5, df30)`。
4. 在 `registry.py` 的 `get_minute_strategies()` 中注册。

## 运行方式不变

盘后日线选股：

```powershell
python main.py
```

盘中实时扫描：

```powershell
python main.py --mode realtime
```

盘中循环扫描：

```powershell
python main.py --mode realtime --loop --interval 60
```

## 注意

本版本主要做策略模块化。分钟数据源仍沿用原来的 `minute_strategy.py` 数据获取流程，也就是 BaoStock 分钟缓存为主。后续如果要改成 Tushare `stk_mins` 或 `rt_min`，建议新增独立的数据加载模块，不要把数据源逻辑写进策略类。


## 缠论分钟级买点策略

本版本已在 `strategies/chanlun_strategies.py` 中新增三个分钟级缠论买点：

```text
缠论一买B点：下跌末端背驰后的反转买点，偏左侧，风险最高。
缠论二买B点：反弹后回踩不创新低，再次启动，确认性比一买强。
缠论三买B点：突破中枢后回踩不回中枢，再次启动，最适合当前主升B点确认。
```

三个策略都已经注册到 `strategies/registry.py` 的 `get_minute_strategies()` 中。
如果后面你想关闭某一个买点，只需要在这里注释掉对应行：

```python
ChanlunFirstBuyMinuteStrategy(),
ChanlunSecondBuyMinuteStrategy(),
ChanlunThirdBuyMinuteStrategy(),
```

建议实盘观察时优先看 `缠论三买B点`；`缠论一买B点` 更偏左侧，不建议单独作为强买点。
